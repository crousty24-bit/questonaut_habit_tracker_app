FAQ

Quick answers for curious users, recruiters, and collaborators.

Questonaut is designed as a production-oriented Rails MVP. This page keeps the most common context easy to scan.

What is Questonaut?

A gamified habit-tracking web app where habits become missions and progress is rewarded with XP, levels, streaks, and badges.

What can users do today?

Users can create an account, manage habits, validate progress, unlock badges, and review statistics from a personal dashboard.

What stack powers the app?

Ruby on Rails 8, Devise, Hotwire, Turbo, Stimulus, Tailwind CSS, SCSS, SQLite, and RSpec.

Is this site the app itself?

No. This GitHub Pages site is a static showcase that presents the product. The Rails application remains separate.

Why a space theme?

The product uses exploration language to make self-discipline feel more engaging: habits become missions and progress feels like a journey.

Is the MVP finished?

The core MVP loop is complete and usable end to end. Future iterations would deepen analytics, templates, and personalization.
