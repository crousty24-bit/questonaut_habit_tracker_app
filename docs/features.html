Features

A focused MVP, built around one end-to-end habit loop.

Questonaut keeps the scope intentional: users can sign up, manage missions, validate progress, earn rewards, and review performance without leaving the core product flow.

Authentication

Simple sign up, sign in, account edit, and password recovery powered by Devise.

  • Onboarding starts from the landing page
  • Protected dashboard once logged in
  • Account recovery stays in scope for the MVP

Missions

Habits are framed as missions so the domain stays playful without losing clarity.

  • Create, edit, and delete habits
  • Daily and weekly frequencies
  • Category-based organization

Validation flow

The validation action is designed to be quick, visible, and repeatable every day.

  • Mission completion logging
  • Streak-aware feedback
  • Single clear action from the dashboard

Gamification

Consistency earns progress. Rewards are intentionally lightweight but motivating.

  • XP gained from key actions
  • Level progression from total XP
  • Unlockable badges and login streaks

Dashboard

The main product surface gathers everything a user needs to understand current momentum.

  • XP and level progress
  • Mission groups and filters
  • Recent badges and weekly summary

Statistics

Mission history becomes readable through charts, counts, and detailed analysis.

  • Success rate and mission days
  • Best streak and badge totals
  • Weekly progress and category distribution

Tech Stack

Built to stay understandable.
